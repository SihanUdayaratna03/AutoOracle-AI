# AutoOracle-AI

A Machine Learning project for predicting used car prices based on various features like age, mileage, and transmission type.


## Dataset
The dataset contains features such as:
- Selling Price
- Present Price
- Kms Driven
- Fuel Type
- Seller Type
- Transmission
- Owner


## Models Used
The following models are evaluated:
1. Linear Regression
2. Lasso Regression
3. Random Forest Regressor


## Setup Instructions
1. Clone the repository.
2. Run `pip install -r requirements.txt`
3. Open the Jupyter Notebook to run the predictions.
